### 확정된 설계 결정 (섹션 ①②③ 사용자 승인 완료)

- **목적**: 전국 공공기관/정부단체/공기업/대학병원의 금고은행 입찰 도래를 한눈에
  보는 인터랙티브 히트맵 지도. v1 범위는 **서울 25구+서울시, 경기 31시군+경기도
  (지자체 ~58) + 서울·경기 주요 대학병원 15~20곳**.
- **데이터원**: 나라장터 기본 + 각 기관 홈페이지 공시/공지/알림 (혼합). 초기엔
  리서치 에이전트 수동 큐레이션으로 정적 데이터 구축, 갱신 파이프라인은 범위 외.
- **형태**: 정적 단일 HTML + 데이터 분리. `dashboard/index.html` +
  `vendor/d3.v7.min.js`(로컬 번들) + `geo/{region}.js` + `data/institutions.js`.
  `file://` 더블클릭 동작을 위해 JSON이 아닌 **JS 전역변수 래핑**(fetch CORS 회피).
- **기술**: **D3 채택**(라이브 데모 비교 후 사용자 선택 — d3-zoom 플라이투의
  "붕 떴다 내려앉는" 느낌 + 휠/드래그 제스처. 오프라인 번들 필수).
- **색 규칙**: 구간형 5단계 — 6개월↓🔴 / 1년↓🟠 / 2년↓🟡 / 2년+🔵 / 미상⚪.
  색은 저장하지 않고 열람 시점에 `contractEnd - today`로 계산.
- **이중 레이어 문법**: 면(폴리곤) 색 = 지자체 금고만, 그 외 기관 = 유형별 마커
  (병원=원●, 공기업=사각■, 공공기관=삼각▲, 미정의 type=◆ 폴백). "모양=유형,
  색=긴급도" 2축 — 색 혼합 절대 금지. 유형 토글 4종(데이터 없으면 비활성),
  8개+ 밀집 시 숫자 뱃지 클러스터.
- **스키마**: `type: 지자체|대학병원|공공기관|공기업`, `region`은 enum 잠금 없이
  17개 시도 코드 문자열(전국 개방) — 렌더러는 `geo/*.js`가 로드된 지역만 그리고
  나머지 기관은 "범위 외" 목록으로. `confidence: 확정|추정|미상`(추정=빗금 표시),
  `sources[]` 1개 이상 필수 — 조작 금지 원칙 그대로 적용.
- **화면**: 최상위 = 좌 서울 ‖ 우 경기 분할 + 상단 공용 범례 + 임박 TOP 5 티커.
  지역 클릭 = D3 플라이투(750ms)로 **지도 2/3 + 우측 임박순 랭킹 패널 1/3** 전환
  (패널 카드 ↔ 마커 상호 강조, 카드 클릭 시 출처 링크 상세). 복귀 = 배경 클릭
  또는 브레드크럼. 줌 전환은 "확!" 하는 강한 애니메이션이 필수 요구.
