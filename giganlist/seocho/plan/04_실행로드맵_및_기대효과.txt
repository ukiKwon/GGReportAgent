# 실행 로드맵 및 기대효과

=================================================
1. 연차별 실행 로드맵 (2026년 하반기 제안 기준)
=================================================

[2026년 하반기 - 착수/기획]
- IT-1 예산공개플랫폼: 서울시 열린재정 포털 API 연동 가능성 검토, 47건 사업의
예산 매핑 마스터데이터 설계 착수
- IT-2 SOC 정보공개 대시보드: AICT벨트(지구지정 완료) 진행단계 데이터부터 우선
반영, 지하화 공약은 "확정 전" 상태 표기 틀만 우선 설계
- IT-3 민원 대시보드: 국민신문고 연계 구조 및 접수번호 체계 조사
- FN-1 SOC 인접지역 지원금: AICT벨트 지구지정 구역 내 임차인·소유자 실태조사

[2027년 - 구축/1차 시행]
- IT-1: 예산공개플랫폼 베타 오픈, 확인 가능했던 사업부터 순차 반영
- IT-2: AICT벨트 진행현황 페이지 1차 오픈
- IT-3: 부서정보 복원 대시보드 1차 구축
- FN-1, FN-2: 지원금 신청 접수 개시

[2028년 - 확산/고도화]
- IT-1: 27개+ 사업 예산 매핑 완성, 목표 확인율 60% 이상 달성
- IT-2: 경부간선도로·반포대로 지하화가 설계 단계 진입 시 대시보드에 반영
- IT-3: 유사 민원 자동 태깅 정례화, 연 1회 통계 공표
- FN-3: 보육종사자 처우개선 보완금 1기 운영

[2029년 - 정착/성과평가]
- IT-1~IT-3 전체 서비스 안정화, 유지보수 체계 전환
- FN-1~FN-3 성과평가 및 차기 계획 반영 검토

2. 핵심성과지표(KPI) 제안
=================================================

[IT-1 예산공개플랫폼]
- 27개 사업 중 예산 확인 가능 비율: 현재 0% -> 목표 60% 이상
- 월간 이용 세션 수, 사업별 페이지 체류시간

[IT-2 SOC 정보공개 대시보드]
- AICT벨트 진행단계 공개 항목 수
- 지하화 공약의 설계 단계 진입 여부 및 반영 소요기간

[IT-3 민원 대시보드]
- 부서별 처리건수·평균 처리일수 공개 항목 수
- 유사 민원 자동 태깅 건수

[FN-1 SOC 인접지역 지원금]
- 지원 신청 건수 및 집행률
- 재건축단지 정보격차 관련 민원(메이플자이 유형) 건수 추이

[FN-2 복합시설 프로그램 매칭지원금]
- 우면열린문화센터 프로그램 참여인원 확대 규모
- 민간 매칭펀드 유치 금액

[FN-3 보육종사자 처우개선 보완금]
- 신규 개원 8곳 수당 적용 완료율
- 아이돌보미 활동 인력 유지율

3. 리스크 및 대응
=================================================

[리스크 1] "47건" 원문 보도자료 전체 목록을 확보하지 못해(spec/07 한계), IT-1
설계 시 27건 외 나머지 20건의 예산 매핑이 누락될 위험
- 대응: 서초구청 공식 보도자료 원문을 직접 요청하거나, 공보관실 협조로 전체 47건
목록을 우선 확보한 후 IT-1 설계에 반영

[리스크 2] 경부간선도로·반포대로 지하화가 아직 확정되지 않은 공약 단계에서 IT-2를
설계하면 실제 계획과 맞지 않을 위험
- 대응: 공약 내용이 아니라 "진행상황을 추적하는 표준 틀"만 먼저 설계, 실제 설계안은
확정 즉시 반영

[리스크 3] FN-1이 AICT벨트·지하화 인접지역에 편중될 경우 형평성 이슈
- 대응: IT-2의 표준화된 진행단계 데이터를 기준으로 지원 대상을 객관적으로 산정

4. 종합 기대효과
=================================================
1. 예산 투명성: 27개 사업 중 0건이었던 예산 확인 가능 사업을 확대해 재정자립도
53.2%(서울 3위)라는 강점이 실제 체감되는 행정으로 연결
2. 대형 SOC 공약 정보격차 해소: AICT벨트·지하화처럼 언론 노출은 높으나 진행상황
추적은 어려운 대형 공약에 대한 주민 접근성 제고
3. 민원채널 개선: 국민신문고 통합의 편의성은 유지하면서 종로구 수준의 부서별
투명성을 보완적으로 회복
4. 예산 효율성: 신규 대규모 시스템 구축보다 기존 인프라(서울시 열린재정 포털,
국민신문고) 재활용 중심으로 설계해 총 사업비를 서초구 2026년 예산(9,417억원)
대비 약 0.14~0.22% 내외로 억제
5. 보육·복지 정책 일관성: 이미 신설된 조리사·보육도우미 수당, 우면열린문화센터
등 확인된 정책의 자연스러운 확장을 통한 정책 일관성 유지
