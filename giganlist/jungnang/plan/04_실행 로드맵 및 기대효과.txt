# 실행 로드맵 및 기대효과

=================================================
1. 연차별 실행 로드맵 (2026년 하반기 제안 기준)
=================================================

[2026년 하반기 - 착수/기획]
- IT-1 홈페이지 접속안정성 개선: 가장 시급한 선행과제 - 외부 진단 도구로 원인
파악 착수(본 조사에서 확인된 "Parse Error" 현상의 정확한 원인 규명 포함)
- IT-4 공약이행 추적 플랫폼: 중랑동행 비전위원회의 공약 구체화 완료 시점에 맞춰
사전 설계 착수
- IT-2 예산공개플랫폼: 서울시 열린재정 포털 API 연동 가능성 검토
- FN-1 정비사업 지원금: 4개 구역(상봉9·상봉7·중화6·면목10) 실태조사

[2027년 - 구축/1차 시행]
- IT-1: 개선 완료 및 정기 점검 프로세스 가동
- IT-2: 예산공개플랫폼 베타 오픈, 확인된 34개 항목 중 우선 확인 가능했던 사업부터
순차 반영
- IT-3: 정비사업 진행현황 지도 1차 구축(상봉9구역 우선 템플릿 적용)
- IT-4: 중랑동행 비전위 공약 확정안 반영 1차 오픈
- FN-1, FN-2: 지원금·통합패키지 신청 접수 개시

[2028년 - 확산/고도화]
- IT-2: 5대 분야 전체 예산 매핑 완성, 목표 확인율 70% 이상 달성
- IT-3: 4개 정비구역 전체 표준화 완료
- FN-2: 청년 통합패키지 1기 성과평가
- FN-3: 특교금 매칭 복지지원금 1기 운영

[2029년 - 정착/성과평가]
- IT-1~IT-4 전체 서비스 안정화, 유지보수 체계 전환
- FN-1~FN-3 성과평가 및 민선9기 차기 계획(수립 시) 반영 검토

2. 핵심성과지표(KPI) 제안
=================================================

[IT-1 홈페이지 접속안정성]
- 외부 표준 진단 도구 기준 접속 성공률: 현재 사실상 0%(본 조사 기준) -> 목표 100%
- 검색엔진 색인 페이지 수 증가율

[IT-2 예산공개플랫폼]
- 34개 사업 중 예산 확인 가능 비율: 현재 약 6%(2/34, 신뢰도 낮음) -> 목표 70%
이상
- 월간 이용 세션 수

[IT-3 정비사업 진행현황 시스템]
- 4개 구역 중 표준화된 정보공개 구역 비율: 현재 25%(1/4, 상봉9구역만) -> 목표
100%

[IT-4 공약이행 추적 플랫폼]
- 중랑동행 비전위 공약 확정 후 플랫폼 반영까지 소요기간
- 공약별 이행률 공개 항목 수

[FN-1 정비사업 지원금]
- 지원 신청 건수 및 집행률(4개 구역별)

[FN-2 청년 통합패키지]
- 통합 상담 이용 청년 수, 응시료지원->창업상담 전환율

[FN-3 복지매칭지원금]
- 특교금 매칭 재원 확보액, 복지사업 대상자 확대 규모

3. 리스크 및 대응
=================================================

[리스크 1] IT-1의 원인(Parse Error)이 본 조사에서 확인한 도구(WebFetch) 특유의
한계일 뿐, 실제 구민이 사용하는 일반 브라우저에서는 문제가 없을 가능성
- 대응: 착수 단계에서 반드시 외부 표준 웹 진단 도구(예: W3C 검증도구, 크롤러
시뮬레이터)로 재검증 후 실제 문제 여부를 확정하고, 문제가 없다면 IT-1 예산을
축소하고 IT-2~4에 재배분

[리스크 2] FN-1이 민원게시판 데이터 없이 설계되어 실수요 파악이 부정확할 위험
(spec/08의 근본적 한계)
- 대응: IT-1 개선 이후 민원게시판이 실제로 열람 가능해지면, 그 시점에 FN-1의
지원 대상·규모를 재검토하는 것을 필수 절차로 명시

[리스크 3] 중랑동행 비전위원회의 최종 공약이 IT-4 설계와 맞지 않을 위험
- 대응: 종로구 제안과 동일하게, 공약 내용이 아니라 "공약 이행을 추적하는 표준
틀"만 먼저 설계하고 실제 데이터는 확정 즉시 입력

4. 종합 기대효과
=================================================
1. 디지털 접근성 회복: 홈페이지 자체의 접속 장애(spec/00·08에서 실제 확인된
문제)를 해소해 이후의 모든 디지털 서비스가 정상 작동할 수 있는 전제조건 확보
2. 예산 투명성: 34개 사업 중 32개 이상이 예산 확인 불가능했던 구조적 문제(spec/07)
해소
3. 정비사업 정보격차 해소: 4개 정비구역(상봉9·상봉7·중화6·면목10) 간 정보공개
편차를 해소하고 표준화된 안내체계 마련
4. 정책 연속성 위의 투명성 내장: 3선 연임 구청장의 정책 연속성이라는 강점 위에,
중랑동행 비전위원회의 신규 공약 구체화 시점을 활용해 투명성·추적성을 처음부터
반영
5. 재정 전략 계승: 재정자립도 열세를 특교금 확보(서울시 1위)로 보완해온 중랑구의
기존 전략을 복지분야 매칭재원으로 명시적으로 확장
