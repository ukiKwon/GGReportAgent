[구청장에게 바란다 - 운영 방식(홈페이지 안내문으로 확인)]
- 게시글은 "처리완료 전까지 비공개로 등록"되고, 담당자 답변 완료 후 공개/비공개로
전환됨(종로구 "답변완료 후 공개"와 유사한 운영방식)
- 비공개 게시글 답변은 [나의 민원 보기]에서 확인, 답변은 홈페이지 및 휴대전화
장문메시지(MMS)로 전송
- 게시 규칙: 구정 무관/비방·명예훼손/욕설·저속표현/정치적 목적/광고/동일안건 반복
게시글은 비공개 전환 또는 삭제될 수 있음

[중요 - 2026년 실제 게시글 목록 확인 결과]
구청장에게 바란다 게시판의 2026년 실제 게시글 목록(제목·작성일·담당부서·답변상태)은
본 조사 환경에서 확인하지 못함(확인안됨). 사유:
- 게시판이 JavaScript 동적 렌더링 방식이라 정적 페이지 수집(WebFetch)으로는 게시글
목록 테이블이 로드되지 않음(HTML에 메뉴/네비게이션만 존재, 목록 미포함)
- 상당수 게시글이 "처리완료 전 비공개" 운영 원칙상 목록 단계에서 비공개일 가능성
- 이는 종로구 옴부즈만 게시판(로그인 필요로 확인 불가)과 유사한 접근성 한계임
※ 종로구는 "구청장에게 바란다" 게시판 목록이 정적으로 노출되어 15건을 전수 확인할 수
있었으나(종로 spec/08), 구로구는 동적 렌더링으로 동일 방식의 목록 확인이 불가했음.
이는 구로구 게시판이 종로구보다 폐쇄적이라는 뜻이 아니라 "조사 도구(정적 수집)의
한계"이며, 실제 접근성 우열은 이 조사만으로 단정할 수 없음(부분확인)

[민원 유형 추정 - 지역 특성 기반, 직접 확인 아님에 주의]
아래는 게시글 원문 확인이 아니라 spec/02~05에서 확인된 지역 현안으로부터 "예상되는"
민원 유형이며, 실제 게시글로 검증된 것이 아님(추정):
1. 재개발·재건축 관련: 가리봉1구역/고척동253/개봉·구로동 신통기획(spec/04) 등 대규모
정비사업 진행 중 - 정비구역 진행상황·이주·분담금 관련 민원이 예상되나 실제 게시글
미확인
2. 교통/주차: 신도림역 구로1유수지 공영주차장 신규 조성(spec/04, 3-6)은 주차난
민원 대응 성격으로 해석 가능 - 실제 주차 민원 게시글은 미확인
3. GTX-B 환기구: 경인고 인근 환기구 설치 문제(spec/01)는 주민 반발 현안으로 구청장이
"주민 편 대응" 입장을 밝힌 사안 - 관련 민원 존재 개연성 있으나 게시판 원문 미확인
4. G밸리 배후 생활인프라: G밸리 종사 청년·1~2인 가구의 주거·생활편의 관련 요구
(spec/03) - 미확인

[역사적 배경 참고 - 2026년 데이터 아님]
구로타임즈 과거 기사(구민체육센터 명도 관련 민원 임의삭제 논란)가 검색되나, 이는
2006~2007년경의 과거 사안으로 2026년 현황과 무관함. 2026년 민원 데이터로 오인하지
말 것(본 조사에서 시점을 교차확인하여 제외 처리함)

운영으로 정적 수집이 불가했음. 브라우저 렌더링(headless) 도구로 재접속하거나 로그인
