# 마포구 5대 분야 18개 사업/항목 검색 확인 결과

조사일 기준: 2026년 7월 20일

[방법론]
종로구 조사와 마찬가지로, 마포구도 조사 시점(2026.7.20) 기준 민선8기->9기 전환
초기(취임 2주)라 신임 구청장의 공식 "4개년계획"이 없음(spec/01). 다만 마포구는
"2026년 주요업무계획"이라는 문서 자체는 홈페이지에 실재하여, 이를 실제로 다운로드
받아 pdftotext로 텍스트 추출을 시도했으나 폰트 인코딩 문제로 한글 본문 추출에
실패함(spec/01 상세 기술). 따라서 본 조사는 spec/02~06에서 실제 언론보도·공식
홈페이지·마포옴부즈만 게시판 검색으로 확인한 18개 사업/항목 전체에 대해, "몇 개의
독립 출처에서 교차 확인되는지"를 재검토하는 방식으로 확인등급을 부여함(전수 조사,
표본 아님 - 18건 자체가 검색으로 도출된 목록이므로 별도 표본추출 과정이 없었음).

[분류 기준]
- 확인됨: 구체적 일자·수치(진행단계, 규모, 승인일 등)가 명시된 채로 뉴스 보도되거나
마포구청 공식 홈페이지에서 직접 확인됨
- 부분확인: 사업/방향성의 "존재"는 확인되나 세부 내용(정확한 예산, 시행시기,
사업명 확정 여부 등)이 불명확하거나, 단일 출처/공약 수준의 언급에 그침
- 확인안됨: 문서 존재는 추정되나 구체적 내용을 특정할 수 없는 경우

[한계 및 후속 작업 제안]
1. "2026년 주요업무계획" PDF(156페이지, mapo.go.kr "마포소식>주요업무계획"
게시판에서 실제 다운로드 확인)는 종로구 사례와 달리 이미지 임베딩이 아닌 정상
텍스트 레이어 PDF였으나, 본문 폰트가 ToUnicode CMap 없는 서브셋 폰트라 한글
추출이 되지 않음. 반면 예산으로 추정되는 숫자(13,700/1,360/41,552/12,700/
109,795/3,180/90,702/410,864 등, 단위 불명)와 날짜, 목차 구조(5대분류x3소분류)
는 추출에 성공함 - 즉 "사업명은 모르지만 개별 사업 예산 수치 자체는 문서 안에
존재한다"는 사실은 확인됨. 후속 작업 시 해당 PDF를 발간한 부서(추정: 기획
예산과)에 원본 한글(HWP) 파일을 요청하거나, OCR 도구(Tesseract 한국어팩 등)를
갖춘 환경에서 이미지 렌더링 방식으로 재시도할 필요가 있음.
2. 재정자립도(%) 수치는 다수 검색 시도에도 확인하지 못함(spec/01) - 서울시
열린데이터광장 등 통계 포털에서 직접 조회하는 후속 작업이 필요함.
3. "확인안됨"이 사업의 부재를 의미하지 않음 - 특히 3분야(문화관광)의 부분확인
3건은 모두 취임 2주 시점의 공약 수준 서술이라 아직 사업 문서화가 되지 않았을
가능성이 높음.
4. 예산액이 구체적으로 확인된 사업/항목은 18건 중 5-3(주민참여예산 공모사업,
10억원) 1건뿐이며, 이마저도 복지 분야 전용이 아닌 7개 분야 공통의 공모 규모임.
개별 사업 단위의 예산 확인율은 종로구(1/33)와 유사하게 매우 낮은 수준임.
