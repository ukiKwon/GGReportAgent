# 서대문구 5대 분야 24개 사업/항목 검색 확인 결과

조사일 기준: 2026년 7월 20일

[방법론 - 종로구 조사와의 차이점]
종로구 조사는 WebSearch를 전량 사용해 다수 매체 교차확인을 수행했으나, 서대문구
조사는 착수 직후 세션의 WebSearch 예산이 소진되어(spec/00 참고) 이후 전량
WebFetch(duckduckgo html 검색결과 페이지 + 개별 URL 직접조회)로 전환함. WebFetch는
검색엔진 결과를 1회 호출로 요약받는 방식이라, 종로구 조사처럼 "몇 개의 서로 다른
언론사 URL을 각각 열어 대조"하는 방식이 아니라 "요약 결과 안에 몇 개 출처가
언급되는지"를 기준으로 확인등급을 매김. 이로 인해 종로구 대비 개별 사업의 교차
확인 강도가 전반적으로 낮게 나올 수밖에 없는 구조적 차이가 있음을 먼저 밝힘.

[분류 기준]
- 확인됨: 2개 이상 서로 다른 출처(언론사/공식페이지)에서 교차 확인되거나, 구체적
수치(날짜·금액·세대수 등)가 반복 조회에서도 일관되게 재현됨
- 부분확인: 1개 출처에서만 확인되거나, 사업의 "존재"는 확인되나 세부내용(예산·
현재 진행상태)은 확인되지 않음
- 확인안됨: 후보 시절 공약 수준에 그치거나, 재조회 시 관련 내용이 나오지 않거나,
서로 다른 조회에서 상충되는 내용이 나온 경우

24건 전체에 대해 확인 시도함(전수, 표본 아님).

=================================================

13개동, 79,000㎡)이나 복수 출처 대조 못함

1-3 홍제역세권활성화/유진상가·인왕시장 확인됨 - "신촌·이대 재개발" 검색과
"유진상가·인왕시장" 검색 두 개의 독립적 질의에서 49층, 4.25만㎡대, 자치구 최초
사업시행자라는 동일 수치가 재현되어 교차확인으로 인정

vs 38층/3,026세대)하여 동일사업 여부를 확정하지 못함. 두 수치 병기 후 확인안됨
처리(spec/02 참고)
1-5 홍제4구역/홍은13·15구역 부분확인 - 단일 출처, 구역별 진행단계 서술 수준
1-6 서부경전철·강북횡단선 부분확인 - 구청장 공약 언급 수준, 서울시 사업이라
서대문구 단독 진행상황 확인안됨

- 확인됨: 6건 (25%)
- 부분확인: 12건 (50%)
- 확인안됨: 6건 (25%)

[분야별 확인율 순위 (확인됨 비율 기준)]
1위 - 4분야 문화관광·환경: 2/4 확인 (50%)
2위 - 1분야 도시재생·주거환경, 2분야 경제일자리·청년상권, 3분야 복지건강,
5분야 안전스마트행정: 각 1건씩 확인(각각 17~25% 수준, 표본수가 작아 순위 큰
의미 없음)

[한계 및 후속 작업 제안 - 종로구 대비 서대문구 조사의 고유 한계]
1. WebSearch 예산 소진으로 조사 전 과정을 WebFetch(검색결과 페이지 요약)로
수행해, 종로구 조사(다수 매체 원문을 개별적으로 열어 대조)보다 교차확인 강도가
구조적으로 낮음. 이는 확인율 수치를 종로구(39%)와 직접 비교하기 어렵게 만드는
요인임.
2. WebFetch 자체의 신뢰성 문제를 이번 조사에서 두 차례 직접 확인함:
(1) 민선8기 4개년계획 페이지(sdm.go.kr/chief/M8_Manifesto/m8_02.do)의 인코딩
오류로 정책분야명 등 텍스트 정보의 신뢰도가 낮았고,
(2) "구청장에게 바란다" 게시판(spec/08)은 동일 URL·동일 게시글 번호에 대해
3회 조회 시마다 전혀 다른 제목이 반환되어 명백한 hallucination으로 판단, 전량
폐기함.
3. 1-3과 1-4 사이의 세대수·층수 불일치(spec/02)처럼, 같은 지역(홍제동)의 여러
정비사업 보도가 혼재되어 사업 간 경계를 명확히 구분하기 어려운 사례가 있었음.
4. 4-4 탄소중립계획 조사 중 타 자치구(은평구) 시설 정보가 섞여 나온 사례가
있어(spec/05), WebFetch 요약 결과를 그대로 신뢰하지 않고 지리적 정합성 등
교차검증을 거쳐야 함을 재확인함.
5. 예산액이 구체적으로 확인된 사업은 24건 중 2건(3-1 행복한밥상 13.6억원, 5-2
주민참여예산 11.5억원)뿐임 - 종로구(1/33)보다는 비율이 다소 높으나(2/24, 약 8%),
여전히 개별사업 예산 투명성은 낮은 수준.
