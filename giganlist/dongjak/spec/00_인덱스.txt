[문서 성격 차이 - 참고]
동작구 최상위 계획문서는 도봉구의 "4개년계획(장/절 구조)"과 달리
"연간 주요업무계획"(2026년판, 40쪽) 형태이며, 세대별(6개) + 분야별(7개) 챕터 구조를
가진다. 각 챕터는 연혁(22~25년 추진사업 회고) + "2026년 중점사업" 상세 항목으로 구성됨.
장기 SOC/개발계획(GTX, 신속통합기획 등)은 "구정주요현황(2026)" PDF 및
도시개발계획포털(ud.dongjak.go.kr), 동작구의회 회의록에서 보완 확인함.

   - 이미지 임베딩 한글 폰트로 텍스트 추출 불가 -> 전 페이지 이미지 렌더링(130dpi)으로 직접 확인

   - 마찬가지로 이미지 임베딩 폰트, 앞부분(기본현황/조직/재정) 위주로 이미지 렌더링 확인
   - 2페이지가 1개 PDF 페이지에 스프레드 형태로 배치된 구조(인쇄페이지 번호 기준 확인 필요)
