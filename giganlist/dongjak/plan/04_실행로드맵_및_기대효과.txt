2. 핵심성과지표(KPI) 제안
=================================================

[IT-1 신규사업 개시 알림 페이지]
- 검색조사 기준 확인안됨·부분확인 비율: 현재 41%(9%+32%, spec/07) -> 목표 15% 이하
- 신규 개관형 사업의 개관일 기준 사전 안내 페이지 노출 리드타임(목표 D-30일 이상)

[IT-2 민원채널 접근성 복구]
- "구청장에게 바란다"·"옴부즈만 민원 신청" 목록 페이지 외부 접근(WebFetch/검색엔진
  색인) 가능 여부 - 현재 완전차단(spec/08) -> 목표 목록 100% 공개
- 알림톡 발송 후 재문의(중복 민원) 감소율

[IT-3 의회 회의록 연계 위젯]
- 대형 개발사업(신속통합기획 16개소, 모아타운 5개소 등) 중 위젯 매칭 완료 비율
- 사업정보 페이지 내 회의록 링크 클릭률(주민 관심도 지표)

[IT-4 생애주기 챗봇]
- 카카오톡 채널 등록자 수, 생애주기 단계별 알림 수신 동의율
- 신청기간 임박 알림 발송 후 실제 신청 전환율(청년·신혼부부 전월세 지원 등 확인됨
  사업 우선 측정)

[FN-1 신규 개관시설 활성화 지원금]
- 청년센터·JUMP1 개관 후 3개월 이용자 수(목표: 유사 시설 평균 대비 상회)

[FN-2 노후주택 안전개선]
- 빌라관리센터 4개 권역 통합창구 신청 건수, 3개 지원사업(담장/축대/공동주택개선) 집행률

[FN-3 청년 디지털 서포터즈]
- 참여 청년 수, 수료 후 청석창업지원센터 입주기업 연계 취업 건수

3. 리스크 및 대응
=================================================

[리스크 1] "구청장에게 바란다"·"옴부즈만 민원 신청"이 완전 차단된 원인이 단순 렌더링
방식이 아니라 개인정보 보호를 위한 의도적 설계일 가능성
- 대응: IT-2는 상세 내용이 아닌 "목록(제목·접수일·처리상태)"만 공개하는 것으로 범위를
  한정하고, 개인정보가 포함될 수 있는 민원 원문은 비공개 유지. 파일럿을 "구청장에게
  바란다" 1개 채널로 한정해 검증 후 확산.

[리스크 2] 동작구의회 회의록(assembly.dongjak.go.kr)은 구청 소관이 아닌 의회사무국
소관 시스템으로, API 연동에 별도 협의·승인 절차 필요
- 대응: IT-3는 전면 API 연동이 어려울 경우 정기 크롤링+수동 검수 방식으로 축소
  운영 가능하도록 단계적 설계

[리스크 3] 대형 개발사업(신속통합기획 16개소, 재정비촉진구역 등)은 서울시·조합·시행사
등 다자 이해관계자가 얽혀 있어 구 자체 정보공개 속도에 한계
- 대응: IT-1·IT-3는 "구 자체 사업 현황"과 "상위기관 연계사업 진행경과"를 구분 표시해
  주민의 기대치를 관리(완공시점 등은 상위기관 확정 전까지 잠정치로 표기)

[리스크 4] FN-1 신규 개관시설 활성화 지원금의 형평성 이슈(먼저 아는 사람만 수령)
- 대응: IT-1·IT-4의 사전 안내(D-30일 배너, 챗봇 알림)와 지원금 지급을 반드시 연동시켜
  "정보를 접한 사람만 신청 가능"한 구조 자체를 개선 대상으로 설계

4. 종합 기대효과
=================================================
1. 신규사업 접근성: 2026년 주요업무계획 95개 중점사업 중 신규 개관형 사업의 "확인안됨·
   부분확인"(41%, spec/07) 문제를 구조적으로 해소해 예산 집행과 주민 체감 간 시차를 축소
2. 민원 채널 회복: 완전 차단 상태였던 핵심 민원채널 2곳(구청장에게 바란다, 옴부즈만
   민원신청)의 최소한의 목록 공개를 통해 대외 평가지표(민원서비스종합평가 우수 등급,
   spec/01)의 근거를 스스로 갱신·증명할 수 있는 데이터 기반 마련
3. 정보 소재 일원화: "구청 홈페이지가 아닌 의회 회의록에서만 확인되는" 구조(spec/07)를
   위젯 연동으로 보완해 주민의 정보 탐색 부담 경감
4. 예산 효율성: 신규 대규모 시스템 구축보다 기존 인프라(동작청년센터, 빌라관리센터,
   청석창업지원센터, 의회 회의록 시스템, 청년정책 전용사이트)를 재활용하는 방식으로
   설계해 총 사업비를 동작구 2026년 예산(9,331.3억원) 대비 약 0.2% 내외로 억제
5. 선순환 구조: IT 사업 실행에 청년 서포터즈(FN-3)를 연계해 별도 인건비 부담 없이
   청년 일자리 창출과 사업 실행력 확보를 동시에 달성, 재정자립도 28.87%(spec/01)라는
   재정 여건 하에서도 지속가능한 실행 모델 제시
