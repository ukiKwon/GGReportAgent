동작구청 홈페이지 민원 관련 게시판 - 2026년 게시글 조사 결과
=================================================
조사일 기준: 2026년 7월 16일
조사 대상: 종합민원/참여소통 메뉴 하위 민원 관련 게시판 전체(사전 확보된 href 목록 기준)
조사 방법: WebFetch 도구로 각 게시판 URL 직접 접근

[전체 구조 파악]
사전 확보된 href 목록 중 다수가 실제 게시판이 아니라 "안내페이지"(정적 정보 제공)이거나
JavaScript 렌더링 방식(SPA형 목록)이라 WebFetch로 목록 텍스트를 가져올 수 없는 경우가 많음.
WebFetch로 실제 게시글 목록을 확인할 수 있었던 것은 아래와 같이 제한적임:
- 자유게시판(정상 확인)
- 칭찬합시다(정상 확인)
- 구민제안(공모제안 안내 확인, 개별 게시글 목록은 페이지수만 확인)
- 옴부즈만 자료실(정상 확인)
- 주민참여예산 사업제안(안내 페이지만 확인, 개별 게시글 목록 미확인)
- 민원상담(안내 페이지만 확인)
- 옴부즈만 안내(안내 페이지만 확인)

WebFetch로 빈 페이지("이 웹페이지는 빈 페이지입니다")로 반환되어 확인 불가능했던 것:
- 구청장에게 바란다 (JS 렌더링 방식으로 추정 - 도봉구 조사에서도 동일 패턴 확인된 바 있음)
- 옴부즈만 민원 신청 (동일하게 JS 렌더링 방식으로 추정)

   연계 안내만 있고 자체 게시판은 별도로 없는 구조로 추정.

8. 구청장에게 바란다 (/portal/singl/myPage/appmst114List.do?pReqstCd=APPMST114&menuNo=201504)

   확인안됨 - JavaScript 기반 SPA 렌더링 페이지로 추정되며, 정적 HTML을 가져오는 WebFetch
   방식으로는 게시글 목록을 확인할 수 없었음. 실제 게시글 존재/제목/처리상태는 확인안됨.

9. 옴부즈만 민원 신청 (/portal/singl/myPage/appmst061List.do?menuNo=201321)
   접근: WebFetch로 시도했으나 "빈 페이지"로 반환됨.
   확인안됨 - 8번과 동일하게 JS 렌더링 방식으로 추정되며 로그인 필요 가능성도 있음
   (도봉구 조사에서도 옴부즈만 민원 신청은 로그인 후에만 상세조회 가능했던 점과 유사).

    확인안됨 - 이번 조사에서 동작구청 메인 페이지에서 국민신문고로 직접 연결되는 링크는
    별도로 확인/시도하지 않았음. 민원상담 안내페이지에서 "국민신문고와 연계"한다는 서술만
    확인됨(6번 항목 참고). 추측하여 단정하지 않음.

[전체 요약]
1. WebFetch로 실제 게시글 목록을 확인할 수 있었던 게시판은 자유게시판, 칭찬합시다,
   옴부즈만 자료실 3곳이며, 나머지는 안내페이지 확인에 그치거나(구민제안, 주민참여예산,
   민원상담, 옴부즈만안내) JS 렌더링으로 완전히 접근 불가(구청장에게 바란다, 옴부즈만
   민원신청)했음.
2. 2026년 실제 게시글이 확인된 곳(자유게시판·칭찬합시다·옴부즈만자료실)에서는 모두
   활발한 게시 활동이 확인됨(자유게시판 754건/76페이지, 칭찬합시다 1,871건/188페이지).
3. 도봉구 조사와 비교 시, 동작구는 "구청장에게 바란다"와 "옴부즈만 민원신청"이 도봉구
   사례(모바일/JS 전용 페이지라 목록 열람에 제약)와 유사한 접근성 한계를 보임. 이는
   두 자치구 공통으로 새올전자민원창구·SPA 프레임워크 기반 게시판이 구글 크롤링이나
   단순 WebFetch로는 콘텐츠 확인이 어려운 구조적 특징으로 추정됨.
4. 개인정보 보호: 주민참여예산 사업제안, 구청장에게 바란다, 옴부즈만 민원신청 등은 본인
   로그인 후에만 상세내용 열람 가능한 구조로 추정(직접 확인은 안됨, 도봉구 사례 유추).
5. 이번 조사에서는 "2026년 민원 유형 패턴"(예: 특정 교통/보행 민원이 다수 차지 등)을
   도봉구 수준으로 구체적으로 도출하지 못함 - 이는 핵심 민원게시판(구청장에게 바란다)에
   대한 WebFetch 접근이 실패했기 때문이며, 후속 조사에서는 브라우저 자동화 도구(헤드리스
   브라우저 등)를 활용한 재시도가 필요함.

[후속 작업 제안]
- 구청장에게 바란다, 옴부즈만 민원신청 게시판은 실제 브라우저 렌더링이 가능한 도구
  (예: 헤드리스 브라우저, Playwright 등)로 재조사 필요
- 주민참여예산 사업제안의 개별 제안 게시글 목록 페이지를 별도로 탐색해 2026년 실제 제안
  건수 확인 필요
- 국민신문고 연계 여부는 동작구청 메인페이지 푸터/링크를 직접 확인하는 후속조사 필요
